## Features

- Forms as json
- Dynamic Fields
- Conditional Logic
- User friendly builder
- Internationalization (i18n)
- Field-level validation
- Custom Validation
- Dependent Validation
- Wizards and multi-step forms
- Boostrap 5 friendly
